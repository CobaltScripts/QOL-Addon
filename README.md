The **QOL Addon** is designed to enhance your Minecraft experience by improving various aspects of gameplay with small but powerful quality-of-life tweaks.

This addon is fully compatible with [Cobalt](https://github.com/CobaltScripts/Cobalt) and is written in **Kotlin**.

## Features

- AutoSprint
- AutoHarp
- NickHider (Name Protect)

## Installation

1. Download Cobalt from [here](https://github.com/CobaltScripts/Cobalt).
2. Place the QOL Addon `.jar` file in the `/config/cobalt/addons` folder within your `.minecraft` installation.
3. Launch Minecraft!
